# How to Run Streamlit App

Follow these steps to run the Streamlit app locally:

1. Open the terminal in VSCode.
2. Navigate to the project directory:
   ```bash
   cd '<your project directory>'
````

3. Run the Streamlit app:

   ```bash
   streamlit run <your_file_name>.py
   ```

This will start the app, and you can access it in your browser at:

```plaintext
http://localhost:8501
```

```

---

Replace `<your project directory>` with the location of your project folder and `<your_file_name>.py` with the name of the Streamlit app file you want to run.
